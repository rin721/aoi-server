# メディアプレイヤー

メディアプレイヤーコンポーネントは、動画再生、操作部、キュー、タイムライン、ツールバーを組み合わせ可能な wrapper に分けます。ページはプレイヤー制御ロジックを複製せず、これらのコンポーネントで再生体験を組み立てます。
